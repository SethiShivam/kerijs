# KERI IMPLEMENTATION

This library is the js version of keri .


## How to run test cases

Step:1

Go inside folder 
tests -------> core

Run command:
~~~~
node  test_coring.js
~~~~

To do Blake3 testing : 

Edit test_coring.js file and replace **testslibsodium()**  with **blake()**

Save file and than run Command

~~~~
node  test_coring.js
~~~~
